# Comfydex 0.8 Gallery And Batch UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development when available, otherwise use superpowers:executing-plans. Execute each milestone, review it, and continue automatically when it passes.

**Goal:** Deliver Comfydex `0.8.0` with desktop asset gallery, batch task UI, report preview, comparison UI, safe cleanup UI, bridge operations, Tauri commands, documentation, and release metadata.

**Architecture:** Extend the existing `desktop_bridge.py` and Tauri command bridge. Keep SQLite, path safety, asset mutation, cleanup planning, report generation, and batch parsing in Python/shared modules. Extend React views through `desktop/src/lib/api.ts`.
