# 2.0 Release Checklist

Comfydex `2.0.0` is the Conversational Workflow System release.

## Required Checks

- Run the full Python test suite.
- Run `scripts/validate_release_package.py`.
- Run `npm run typecheck`.
- Run `npm run build`.
- Run `cargo check`.
- Confirm `comfy_20_readiness_report` returns `ready_for_2_0`.

## Scenario Coverage

- Confirm text-to-image is ready.
- Confirm image-to-image is ready.
- Confirm portrait is ready.
- Confirm character consistency is ready.
- Confirm product image is ready.
- Confirm ControlNet is ready.
- Confirm inpainting is ready.
- Confirm upscaling is ready.
- Confirm background replacement is ready.

## Safety Boundary

- Confirm Comfydex performs no automatic downloads.
- Confirm model and custom node gaps use install plans that require confirmation.
- Confirm accepted and rejected install decisions are auditable.

## Publish

```powershell
git tag -a v2.0.0 -m "Comfydex 2.0.0"
git push origin main v2.0.0
git ls-remote origin refs/heads/main refs/tags/v2.0.0 "refs/tags/v2.0.0^{}"
```
