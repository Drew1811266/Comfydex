# Comfydex 1.9 Release Checklist

Comfydex `1.9.0` is the 2.0 Readiness Gate release.

## Required Gates

- Run the full Python gate with `python -m pytest -q`.
- Run `tests/test_readiness.py` to validate the 2.0 Readiness Gate report.
- Confirm `comfy_list_20_scenarios` returns every first-class scenario.
- Confirm `comfy_20_readiness_report` returns `readiness_version` `1.9.0`.
- Confirm Desktop Settings displays the 2.0 Readiness Gate panel.
- Run `python scripts/validate_release_package.py`.
- Run `npm run typecheck`.
- Run `npm run build`.
- Run `cargo check`.
- Confirm version files report `1.9.0`.

## Browser Verification

- Open the desktop app at the local Vite URL.
- Open Settings.
- Confirm the 2.0 Readiness Gate panel shows scenario totals.
- Confirm all first-class scenarios are visible.
- Confirm desktop and mobile viewports have no horizontal overflow.

## Release Commands

```powershell
git tag -a v1.9.0 -m "Comfydex 1.9.0"
git push origin main v1.9.0
git ls-remote origin refs/heads/main refs/tags/v1.9.0 "refs/tags/v1.9.0^{}"
```

## Non-Goals

- Do not claim final 2.0 completion from this release alone.
- Do not silently download models or custom nodes.
- Do not replace ComfyUI's canvas or execution engine.
- Do not add a full visual workflow editor to Desktop.
