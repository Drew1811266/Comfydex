# Comfydex 1.6 Release Checklist

Comfydex 1.6 ships the UI Graph Builder for generated UI workflow JSON, generated graph history, Live Bridge push integration, and the Generated Graphs desktop view.

## Automated Gates

- `python -m pytest -q`
- UI graph builder tests: `tests/test_ui_graphs.py`
- MCP UI graph tools: `tests/test_server_tools.py`
- desktop bridge operations: `tests/test_desktop_bridge.py`
- Generated Graphs desktop view: `desktop/src/views/GeneratedGraphsView.tsx`
- docs/version checks: `tests/test_docs_examples.py tests/test_version.py`
- `python scripts\validate_release_package.py`
- `npm run typecheck`
- `npm run build`
- `cargo check`

## Release Criteria

- `comfy_build_ui_workflow` returns readable UI graph JSON without saving.
- `comfy_generate_ui_workflow` saves a generated UI workflow and appends generated graph history.
- `comfy_generate_push_ui_workflow` saves and performs a Live Bridge push for supported generated UI graphs.
- `comfy_read_ui_graph_history` returns newest generated graph history first.
- The Python desktop bridge supports `build_ui_workflow`, `generate_ui_workflow`, `read_ui_graph_history`, and `push_ui_workflow`.
- The desktop Generated Graphs view shows generated UI workflow history and can push selected records.
- The release keeps no full visual editor, no arbitrary unknown-node graph generation, no automatic downloads, and no automatic custom node installation.
- version files report `1.6.0`.

## Integration

After all gates pass, merge the feature branch into `main`, push `main`, create annotated tag `v1.6.0`, push the tag, and verify:

```powershell
git ls-remote origin refs/heads/main refs/tags/v1.6.0 "refs/tags/v1.6.0^{}"
```
